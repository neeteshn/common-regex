# Common Regex List



| Regex  | Usages |
| ------------- | ------------- |
| /\{(.+?)\}/gm   | This is used to extract any string that starts with *'{'* and *'}'* |
| /(?<=this is)(.*)(?=sentence)/ig   | This is used to extract any string in between *this is* and *sentence*  |
| /<[^>]*>/ig | matches anything enclosed between *< and >* |
